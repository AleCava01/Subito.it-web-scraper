# Subito.it-web-scraper
Simple python web scraper to retrieve ads from Subito.it and convert results to csv

Libraries used:
- Pandas
- PyQt5 (for the GUI)

PyInstaller has been used to generate executable file (dist>Subito_scraper_GUI.exe)
